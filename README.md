# metamorphose

Ce projet est un petit jeu 2D en Java mettant en scène une chenille évoluant dans un monde composé de blocs.
Le joueur peut se déplacer, sauter, et interagir avec les obstacles, le tout avec un système de gravité et une gestion des collisions.
Le niveau est construit à partir d’une matrice de tuiles où chaque valeur indique la présence ou non d’un bloc.

Objectif principal

Développer un moteur de jeu simple permettant d’expérimenter les mécaniques classiques d’un jeu de plateforme :

déplacement horizontal fluide

système de saut avec gravité et hauteur variable

détection de collisions avec plusieurs obstacles

rendu en pixel-art

architecture claire (Avatar, Obstacle, Carte, etc.)

interaction avec des ennemis

système de dégâts et mort du personnage

changement de niveau / passage de phase

objets interactifs et éléments de décor

animations supplémentaires pour la chenille
